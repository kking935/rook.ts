# Rook.io
An online multiplayer game based on the Kentucky-style of the classic southern card game, Rook.
Built with Node.js and Socket.io.

## Installation
```
git clone https://github.com/kking935/Rook.io.git
cd Rook.io
npm install
```

## Running multiplayer server
```
npm start
```
